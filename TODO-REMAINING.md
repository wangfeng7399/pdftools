### 快速修复API错误消息

可以使用以下命令批量替换（谨慎使用）：

```bash
# 在app/api目录下查找所有包含中文的文件
find app/api -name "*.ts" -type f -exec grep -l "错误\|失败\|不存在" {} \;

# 然后逐个文件手动替换，确保上下文正确
```
